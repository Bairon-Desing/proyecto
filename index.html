Acerca De

Esta versión del lenguaje incluye una serie de elementos para aumentar la accesibilidad y mejorar el resultado final de las páginas. Algunas de estas funciones son: a) añadir una cabecera a la tabla, para señalar su contenido y permitir que los sistemas de voz para discapacitados visuales puedan conocer la información que hay dentro; b) agrupar columnas, lo cual permite producir un código más “limpio”; y c) utilizar las opciones de alineación vertical y horizontal para filas y columnas.
